# 养老平台

这是一个养老服务平台的前端项目，提供以下功能：

- 医生管理界面
- 护士管理界面
- 主任管理界面
- 财务管理界面
- AI 工作流助手
- 护理计划管理

## 技术栈

- HTML5
- CSS3
- JavaScript
- Bootstrap

## 项目结构

```
├── css/                # CSS 样式文件
├── js/                 # JavaScript 文件
├── *.html             # 各个页面
├── care-plan.css      # 护理计划样式
└── care-plan.js       # 护理计划脚本
```

## 如何运行

1. 克隆项目到本地
2. 使用浏览器打开任意 HTML 文件即可查看对应页面
